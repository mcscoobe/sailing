# Product Overview

This is a RuneLite plugin for Old School RuneScape that provides quality-of-life features for the Sailing skill. The plugin enhances the sailing experience with visual overlays, helpers, and automation for various sailing activities including:

- Navigation assistance (rapids, lightning clouds, true tile indicators)
- Facility management (sail trimming, speed boosts, crystal extractors)
- Sea charting helpers (weather stations, current ducks, mermaids)
- Barracuda Trials support (crate highlighting, splits tracking)
- Courier task tracking and visualization
- Trawling assistance (net management, shoal highlighting)
- Salvaging location highlights
- Ocean encounter notifications

The plugin is designed to be highly configurable with extensive user preferences for colors, display modes, and feature toggles organized into logical sections.
